Configuração do Firebase

Para funcionar, você precisa preencher o firebaseConfig em ambos os arquivos (index.html e admin.html) com os dados do seu projeto Firebase.

Acesse firebase.google.com → Seu projeto → Configurações do projeto → SDK e copie o objeto firebaseConfig.

No Realtime Database, configure as regras para: 

{
  "rules": {
    "libnotify": {
      ".read": true,
      ".write": true
    }
  }
}

⚠️ Após configurar, faça deploy dos dois arquivos no Vercel normalmente.
